 Weather AR – інтерактивна зміна погоди в небі

Цей AR-додаток дозволяє дитині навести камеру на небо та перемикати різні погодні умови: сонячно, хмарно, дощ, гроза. Кожна погода супроводжується відповідним звуком (пташки, вітер, дощ, грім).

## Як запустити

1. Відкрийте посилання на GitHub Pages (або запустіть через локальний сервер).
2. Надайте доступ до камери.
3. Наведіть камеру на небо (або будь-яку поверхню, яка буде визначена як небо).
4. Натискайте на кнопки внизу екрана, щоб змінювати погоду та слухати звуки.

## Технології

- [8th Wall](https://8thwall.org) – AR-рушій з відкритим кодом.
- A-Frame – веб-фреймворк для 3D/AR.
- JavaScript – логіка зміни текстур та відтворення звуку.
